# CustomRecyclerViewLayoutType

AndroidのRecyclerViewがよくわからない。

一つのRecyclerViewに複数のレイアウトをどうやって実装するのわからない。

だからサンプルを作った。

Layout1には1つのテキストと中央表示。  
Layout2には2つのテキストと左詰め表示。  
データ作成時に交互になるようにデータを設定した。
